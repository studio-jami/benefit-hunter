# Make Startup Program — Application Archetype

## What this program wants

- **Form shape:** structured: startup stage, product description, industry dropdown, "how will you use Make," referral source, additional comments.
- **Tone:** operational and honest. Make explicitly excludes automation *agencies* — you must read as a product company using Make internally.
- **What wins:** numbered, concrete automation use cases (alerts, syncs, review queues, lifecycle flows). A partner/investor referral can double the operations tier — select the partner in "Who is your Investor?" if you have one.
- **Warning:** duplicate applications are auto-rejected. If you applied before, send a follow-up instead of re-submitting.
