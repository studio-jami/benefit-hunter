# ElevenLabs Startup Grants — Application Archetype

## What this program wants

- **Form shape:** product description, team background, "how will you use ElevenLabs." If the web form misbehaves, the grants team accepts applications by email — same content.
- **Tone:** product-vision-forward. ElevenLabs funds teams for whom **voice is the product surface**, not a feature.
- **What wins:** a vivid description of the voice *experience* you're building, an honest statement of where you are, and a specific list of ElevenLabs capabilities you'll use (Conversational AI, TTS, STT, voice design). Umbrella-pack referral codes can boost the grant — use your own pack's code, never someone else's.

## Templated answers

**Product description**

```
{{company_name}} is {{one_liner}}, with voice as the main interaction layer.

[2–4 sentences painting the voice experience: what does the user hear and
say, what happens in the background, why does voice quality make or break
it? Draw from {{use_case_ai}} / {{use_case_general}}.]

{{description}}

We're honest about where we are: {{product_stage}}, with [what's real today —
repos, docs, working subsystems]. But the direction is firm — this is a
long-term product, not a demo.

ElevenLabs is our planned primary voice layer. Voice quality is the product;
ElevenLabs is the bar we want to build to.
```

**About the team**

```
I'm {{first_name}}, {{role_title}} of {{company_name}}. [2–3 sentences:
how the team ships, what's already built, why the size is a choice not a
limitation.]
```

**How we'll use ElevenLabs**

```
- Conversational AI Agents for [your real-time voice surface]
- Text-to-Speech for [narration/summaries/responses in your product]
- Speech-to-Text in the voice interaction loop
- Voice Design for [branded personas, if applicable]
- [Integration note — e.g. behind a provider adapter so ElevenLabs is the
  default production path]
```

## Notes from a successful application

- The line "voice is core, not a feature" is the eligibility test — if you
  can't say it truthfully, apply to a different program first.
- Honesty about stage, paired with concrete evidence of real work, beat
  inflated traction claims.
- Admiration lands when specific ("we evaluated providers and ElevenLabs
  wins on naturalness and agent tooling") — generic flattery doesn't.
