Benefits Registry Auto-Updater

A small Google Apps Script that reads your own Gmail for startup benefit/credit emails, extracts structured updates with Gemini (your own free API key), and keeps a Registry tab in your own Google Sheet up to date automatically — matched by program name, idempotent, with a full audit log.

Runs entirely inside your Google account. The only outbound call this script makes is to Google's Gemini API, using an API key you generate and control. Nothing is sent to Jami Studio or any third party — we never see your email, your Sheet, or your key.

Quick start

  1. Create (or open) a Google Sheet. Add a tab named Registry (or change CONFIG.REGISTRY_TAB in Code.gs).
  2. In the Sheet: Extensions > Apps Script. Delete the default code.
  3. Create each file listed above and paste in the matching content (open appsscript.json via the gear icon > "Show manifest file").
  4. Project Settings (gear) > Script Properties: add GEMINI_API_KEY with a free key from aistudio.google.com/app/apikey.
  5. In Gmail, create a label benefit-programs and apply it to relevant emails (or change CONFIG.GMAIL_SEARCH_QUERY).
  6. Back in the Apps Script editor, run installTriggers once and authorize the requested Gmail + Sheets permissions.
  7. Optional: import registry.seed.example.csv into your Registry tab to see the expected shape.
  8. Run testSyncNow (or use the "Registry Sync" menu in the Sheet) to verify end to end.
