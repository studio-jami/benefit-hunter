# Tools

Your own one-off maintenance scripts (spreadsheet cleanup, bulk imports,
custom exports, registry rebuilds). None are shipped here on purpose — the
real ones are always tied to your own Sheet ID, GCP project, and service
account, so a generic example wouldn't run as-is for anyone else.

If you're using `../automation/benefits-registry-sync/`, you likely won't
need anything in this folder — `testSyncNow` and the Sheet's own menu cover
day-to-day maintenance.
